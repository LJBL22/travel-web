# travel-web
by pure HTML/CSS/JavaScript

practice points:
1. clip-path effects
2. animated burger ideas
3. click dots (btns) => change slides & contents
   `listener and arrow function`
    ```js
    btns.forEach((btn, i) =>
    btn.addEventListener('click', () => {toggleOpen(i)}))
    ```
5. z-index arrangement: video `000`/ color-opacity `777`/ content `888`/ burger & header `999`
6. video/image as responsive background:
     ```css
     width: 100%;
     height: 100%;
     object-fit: cover;
     ```
8. `.nav-btn:not(:last-child)` to set one-side margin
